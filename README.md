# [WIP] Software Toys PWA

A collection of interactive, installable Progressive Web Apps (PWAs) built with vanilla HTML, CSS, JavaScript, Tailwind CSS, and DaisyUI. Each toy is a focused, magical experience leveraging browser APIs for graphics, sound, and offline support.

* Unified homepage launcher with card/list views (persisted preference)
* Global settings: Theme (emerald/luxury), sound effects, vibration haptics
* Shared UI: Consistent header, controls, tooltips, toasts
* Fully offline-capable PWAs (service workers, manifest)
* No build processâ€”pure vanilla for fast iteration

## How to Run Locally

1. Clone the repo: `git clone https://github.com/J-Tynan/Software-Toys-PWA.git`
2. Open in VS Code
3. Install Live Server extension
4. Right-click any `index.html` > Open with Live Server
5. Install as PWA from browser (Chrome/Edge address bar icon)

## Tech Stack

* HTML5 Canvas / Web Audio API / Web Vibration API
* Tailwind CSS + DaisyUI (CDN)
* Vanilla JavaScript
* Service Workers for PWA
